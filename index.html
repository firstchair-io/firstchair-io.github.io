firstchair.io

The new way to track snowfall is coming.

firstchair.io is an open-source app for tracking snowfall at ski resorts and SNOTEL stations. It's a beautiful, reliable, and ad-free. Always.

In addition to the app, we have started an open data initiative for providing consistent snowfall data for both ski resorts and snotel stations. Hourly updates and historic data are provided with a stable, RESTful API that can be used by all.

Get an email when we're ready for the public.
